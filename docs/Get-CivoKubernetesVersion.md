# Get-CivoKubernetesVersion

## SYNOPSIS
List available Kubernetes versions.

## SYNTAX

```
Get-CivoKubernetesVersion
```

## DESCRIPTION
List available Kubernetes versions you can run on the Civo Cloud Platform.

## EXAMPLES

### EXAMPLE 1
```

```
