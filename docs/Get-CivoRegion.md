# Get-CivoRegion

## SYNOPSIS
Lists all regions.

## SYNTAX

```
Get-CivoRegion
```

## DESCRIPTION
Lists all available regions in the Civo cloud platform.

## EXAMPLES

### EXAMPLE 1
```
Get-CivoRegions
```

## PARAMETERS

## INPUTS

### None
## OUTPUTS

### List of available regions.
